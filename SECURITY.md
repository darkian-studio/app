# Security Policy

Darkian Studio is distributed as a beta APK. We take user safety seriously.

## Supported versions

Only the **latest published beta** on the
[Releases page](https://github.com/darkian-studio/app/releases) receives
security fixes. Older betas are unsupported.

## Reporting a vulnerability

**Do not open a public issue for security vulnerabilities.**

Please report privately using GitHub's
[Private vulnerability reporting](https://github.com/darkian-studio/app/security/advisories/new)
for this repository, or email the maintainers through the contact in the issue
tracker. Include:

- A description of the issue and its impact.
- Steps to reproduce (or a proof-of-concept).
- The DS version and platform affected.

We aim to acknowledge reports within **5 business days** and will coordinate a
fix and disclosure timeline with you.

## Supply-chain trust

- Official APKs are published **only** as GitHub Release assets on this repo.
- The runtime setup script is fetched from
  `raw.githubusercontent.com/darkian-studio/app/main/install.sh`.
- If you obtain DS or its setup script from any other source, treat it as
  untrusted.
