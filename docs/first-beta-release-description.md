# Darkian Studio 1.0.0-beta.1

The first public beta of Darkian Studio — a real IDE on Android, Linux, and macOS.

## What's in this beta
- Editor, terminal, LSP, debugging, and git inside one app.
- One-command runtime setup (`install.sh`), runtime-aware for Termux / Linux / macOS.
- In-app runtime verification during onboarding.
- Check for updates + startup update nudge (GitHub Releases).
- Diagnostics logs and crash reports fully inside the app.

## Install
1. Install Termux from F-Droid: https://f-droid.org/en/packages/com.termux/
2. Download this APK.
3. Complete onboarding, paste the setup command into Termux, then tap Verify setup.

## Known limitations
- `dsterm` remote on Windows is not supported in this beta.
- GitHub Releases only — no Play Store build.
